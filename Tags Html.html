1. <h1>até<h6>

As tags de cabeçalho são usadas para definir títulos e subtítulos em HTML. A diferença entre elas é o nível de importância, com o h1 sendo o mais importante e o h6 o menos.

  • <h1> Título principal ou mais importante.
  • <h2> Subtítulo do título principal.
  • <h3> Subtítulo de h2, e assim por diante até h6, que é o título de nível mais baixo.

2. <p>

  • A tag <p> é usada para definir um parágrafo de texto.

3. <mark>

  • A tag <mark> é usada para destacar parte do texto, geralmente com fundo amarelo, para chamar atenção.

4. <small>

  • A tag <small> é usada para diminuir o tamanho do texto, geralmente usada para notas de rodapé ou texto secundário.

5. <i>

  • A tag <i> é usada para estilizar o texto em itálico. Tradicionalmente, é usada para indicar ênfase ou termos técnicos.

6. <u>

  • A tag <u> é usada para sublinhar o texto.

7. <strong>

  • A tag <strong> é usada para destacar o texto com ênfase, geralmente representado em negrito, mas com um significado semântico de maior importância.

8. <ol>

  • A tag <ol> define uma lista ordenada, ou seja, uma lista numerada.

9. <ul>

  • A tag <ul> define uma lista não ordenada, ou seja, uma lista com marcadores (bolinhas).

10. <li>

  • A tag <li> é usada dentro de <ol> (lista ordenada) ou <ul> (lista não ordenada) para definir um item da lista.

11. <a>

  • A tag <a> é usada para criar links (ancoras) para navegar para outra página ou endereço.

12. <hr>

  • A tag <hr> é usada para inserir uma linha horizontal, geralmente para separar conteúdos.

13. <sub>

  • A tag <sub> é usada para definir texto que deve aparecer em um nível inferior ao texto normal, como em fórmulas químicas ou matemáticas (exemplo: H₂O).

14. <sup>

  • A tag <sup> é usada para definir texto que deve aparecer em um nível superior ao texto normal, como em expoentes (exemplo: x²).

15. <blockquote>

  • A tag <blockquote> é usada para citar um bloco de texto de outra fonte. Geralmente, o texto citado aparece com recuo.
